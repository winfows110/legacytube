# ⚓️ LegacyTube (v1.2 Release)

**「あの日の感動を、もう一度その手に。」**

LegacyTubeは、Android 5.0 (Lollipop) 世代のデバイス（特に Xperia Z2 / Z4 Tablet）で、YouTubeおよびそのライブアーカイブを快適に視聴するために開発されたカスタムプレイヤーです。

> **🚀 [配布サイトでAPKをダウンロードする](https://hukuoka-555.github.io/LegacyTube/)** > ※最新の v1.2 および過去のアーカイブ（v1.0 / v1.1）はこちらから入手可能です。

---

## ✨ 特徴 (Features)

### 🚀 1080p再生の最適化 (Hardware Acceleration)
Android 5.0のWebViewにおいて、GPUハードウェア加速を強制的に有効化。Snapdragon 801等の旧世代SoCでも、高ビットレートなアーカイブ動画の「無限くるくる」を最小限に抑え、滑らかな再生を実現します。

### 🔑 Googleログイン完全対応
独自に調整した「Smart User Agent」を搭載。Googleの最新セキュリティチェックをパスしつつ、YouTubeから「Android向け軽量データ」を受け取る絶妙なバランス（Pixel 10 / Android 10ベース）を構築しました。

### 📺 没入型フルスクリーン (Immersive Mode)
10.1インチの大画面をフルに活用するため、システムバーを完全に隠すイマーシブモードを実装。全画面切り替え時に画面が暗転するバグも v1.2 で解消済みです。

### 🚫 軽量・広告ブロック
ブラウザベースの構造により、アプリ本体のサイズを極限まで軽量化。広告ブロック機能を備え、限られたメモリ（RAM）リソースを動画再生に全振りします。
