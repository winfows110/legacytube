# ⚓️ LegacyTube v1.2
**〜 Xperia Z2 / Z4 Tablet 最適化 YouTube プレイヤー 〜**

Android 5.0 (Lollipop) という制約の中で、2024年8月28日の「あくたん卒業ライブ」などの大切なアーカイブを最高画質で、かつ安定して視聴するために開発されたカスタムWebViewプレイヤーです。

## ✨ 特徴 (Features)
- **🚀 GPU Hardware Acceleration:** WebViewのレイヤーをハードウェア加速に固定。古いSoC(Snapdragon 801等)でも1080p再生時のカクつきを最小限に抑えます。
- **🔑 Smart User Agent:** Googleのログイン制限を回避しつつ、YouTubeからAndroid向け軽量データを受け取る最適なUA（Pixel 10/Android 10ベース）を採用。
- **📺 Immersive Fullscreen:** Android 5.0のナビゲーションバーを完全に隠すイマーシブモードを搭載。10.1インチの大画面をフルに活用。
- **🚫 Ad-Block & Lightweight:** ブラウザベースの構造により、余計なリソースを消費せず軽量に動作。

## 🛠 動作確認済み端末
- **Sony Xperia Z2 Tablet (Android 5.0.2)** - *Main Development Device*
- **Sony Xperia Z4 Tablet (Android 7.0可)**

## 📜 開発の歴史 (History)
- **v1.2 (2026/03/14):** - ログイン安定化、GPU加速の強制有効化、全画面切り替えバグの完全修正。
- **v1.1:** - 全画面モード実装、オリジナルアイコン(レトロテレビ)採用。
- **v1.0:** - プロジェクト始動。Android 5.0での基本動作と広告ブロックを実装。

- **Language:** Java
- **Compile SDK:** 34 (Target: Android 5.0 Lollipop)
